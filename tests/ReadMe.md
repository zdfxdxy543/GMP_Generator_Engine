# tests

English

This folder contains automated checks for parsing, generation, validation, configuration loading, and CLI workflows. The tests are meant to protect the project while the GMP-style template keeps evolving.

中文

这个目录存放自动化测试，用来覆盖解析、生成、校验、配置加载和 CLI 流程。它的作用是保证 GMP 风格模板持续演进时项目行为稳定。

## Coverage / 覆盖范围

- requirement parsing  
- controller file generation  
- external JSON catalog loading  
- KPI weights loading  
- one-command run workflow  
- Simulink placeholder error handling
