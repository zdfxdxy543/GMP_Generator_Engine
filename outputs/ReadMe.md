# outputs

English

This folder stores generated artifacts from CLI runs. It is treated as build output rather than source. The project keeps one smoke-test example here for reference during development.

中文

这个目录存放 CLI 运行生成的产物，属于构建输出而不是源码。当前保留了一份 smoke test 示例，便于开发阶段参考。

## Typical contents / 常见内容

- `generated/`: results from `generate` or `run`.  
- `optimization/`: best candidate JSON from `optimize` or `run`.  
- `run_smoke/`: example output created during local verification.
