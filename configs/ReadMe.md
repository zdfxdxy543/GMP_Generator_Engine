# configs

English

This folder stores editable JSON configuration files used by the CLI. These files let you change the module catalog and KPI weights without touching source code.

中文

这个目录存放 CLI 使用的可编辑 JSON 配置文件。通过它们可以在不改源码的情况下调整模块目录和 KPI 权重。

## Files / 文件

- `module_catalog.json`: candidate CTL modules for each control layer.  
- `kpi_weights.json`: weights for the optimization score.
